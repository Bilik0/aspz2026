# Обчислення довірчих інтервалів (Normal & Student) 
# Функції 
* Обчислення ймовірності для нормального розподілу (Z-score) з використанням `erf` 
* Обчислення ймовірності для розподілу Стьюдента (t-score) методом чисельного інтегрування
* Вивід стандартних інтервалів (90%, 95%, 99%) * Обробка помилок (NaN, Infinity) та валідація вводу
## Програмні вимоги
* Середовище запуску (консоль Windows CMD, PowerShell або Linux термінал)
* Компілятор GCC
* Компіляція .c файлів у виконуваний файл з лінкуванням математичної бібліотеки
## Компіляція 
* Звичайна збірка: `gcc main.c erf_module.c -o stats_prog -lm` З
* Збірка з оптимізацією O2: `gcc -O2 main.c erf_module.c -o stats_o2 -lm` 
* Збірка з оптимізацією O3: `gcc -O3 main.c erf_module.c -o stats_o3 -lm` 
# Запуск коду
`./stats_prog`
